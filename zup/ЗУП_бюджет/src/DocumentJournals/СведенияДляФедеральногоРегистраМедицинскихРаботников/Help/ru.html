Сведения для федерального регистра медицинских работников

Передача данных в “Федеральный регистр медицинских работников” в программе состоит из следующих этапов:

  • Загрузка словарей - классификаторов регистра;
  • Настройка соответствия словарей и данных программы;
  • Подготовка данных для формирования файла выгрузки;
  • Передача xml-файлов на портал федерального регистра.
