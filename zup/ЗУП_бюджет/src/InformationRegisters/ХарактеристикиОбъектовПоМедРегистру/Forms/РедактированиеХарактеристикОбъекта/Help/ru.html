Редактирование характеристик объекта

Настраивается соответствие объектов программы словарям федерального регистра медицинских работников. Каждой характеристике необходимо указать соответствующее значение из словаря медрегистра. Настройка используется для формирования файла выгрузки в регистр.

Если классификатор медрегистра не содержит каких-либо значений, необходимо Загрузить классификаторы. Подробнее о загрузке.

Чтобы охарактеризовать справочники с точки зрения федерального регистра, необходимо в каждом из них нажать кнопку с человечками и указать требуемые характеристики.
