Сведения о медицинских работниках

Документ предназначен для подготовки данных о сотрудниках для выгрузки в федеральный регистр медицинских работников.

Подробнее о передаче данных в Федеральный регистр медицинских работников см. здесь.

В документе на заданную дату заполняются только те данные по сотрудникам, для которых настроено соответствие словарям федерального регистра медицинских работников.

Чтобы воспользоваться кнопкой Заполнить, обязательно должно быть настроено соответствие для должности и подразделения сотрудника.

При необходимости информацию о сотрудниках можно изменить непосредственно в документе, это не повлияет на личные и кадровые данные в самой программе.

После записи документа доступно сохранение сформированного xml-файла на диск. Файл используется для загрузки сведений в федеральный регистр.
